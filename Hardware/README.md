## Install `xtl` and `xtensor`

```
git clone https://github.com/xtensor-stack/xtl
cd xtl
mkdir build
cd build
cmake ..
sudo make install
```

Repeat the same steps for the `xtensor`.


```
git clone https://github.com/xtensor-stack/xtensor
cd xtensor
mkdir build
cd build
cmake ..
sudo make install
```
